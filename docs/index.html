Missouri COVID data

The data on this page was last updated Friday, August 26, 2022. The original data comes from the Missouri Department of Health and Senior Services, where it is only presented in graphical form. The purpose of this project was to capture the data in a more usable format. While I still believe in the original intention, I doubt the value of the project at this point and I'm tired of getting periodic build errors, so I'm turning it off. The data from July 27, 2021, to August 24, 2022, is still available for download below.

Download

This data is available to download as CSV, JSON, or a sqlite database. To read more about the data formats, please check the project README.
